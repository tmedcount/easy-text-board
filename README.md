#### MVC구조 개념의 텍스트 보드 프로그램
  - Model : 어플리케이션의 핵심.
  - View : 사용자가 보는 화면을 의미한다.
  - Controller : 사용자의 요청을 받아서 해석한 후 Model에게 다시 요청한다.
